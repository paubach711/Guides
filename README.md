# Personal Guides

A small collection of personal guides written by me, for me.  
This repository is public in case it helps anyone else.

## About

This repository contains technical notes, installation steps, troubleshooting procedures, and quick guides that I have written while working on different systems and projects.

Most of these guides are based on real setups, real errors, and real fixes, so they are mainly intended as practical references rather than polished official documentation.

## Contents

The repository may include guides about topics such as:

- Linux system administration
- HPC environments
- Lustre
- Networking
- Virtualization
- Storage
- Services and infrastructure
- Troubleshooting notes

## Disclaimer

These guides are provided as-is.

They worked for my specific environment at the time they were written, but they may need changes depending on your system, distribution, kernel version, network configuration, or software versions.

Always review the commands before running them, especially if they require root privileges or modify system configuration.
